# Quiz Planet

Multiplayer trivia for up to 5 players. Pick a category together, race the clock, climb the leaderboard.

**[Play now](https://shrihari1999.github.io/quiz-planet/)**

## Features

- Up to 5 players per lobby, each with their own color (CS2-style: yellow, purple, green, blue, orange)
- Public lobby list — see open games live, no room codes needed
- Vote on category in the lobby (all 24 OpenTDB categories + "All")
- 9 questions per game: 3 easy, 3 medium, 3 hard
- 3-second question reveal, then 15 seconds to answer
- Speed bonus: +5 / +3 / +1 points for answering in the 1st / 2nd / 3rd second
- Live timer markers showing when each player answered
- Color-coded name pills on each option after the reveal
- Podium and play-again at game end
- Installable as a PWA
